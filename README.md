# Projeto-CRUD
Projeto CRUD simples de gerenciamento de alunos usando:
 - PHP (8.1), 
 - MySQL,
 - Javascript,
 - JQuery, 
 - Ajax,
 - Bootstrap 5, 
 - HTML5 e 
 - CSS.
